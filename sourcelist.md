## Aggregation Philosophy

1. **Quality over quantity** — Only the best, most unique content from each source
2. **No duplicates** — If two sources have similar skills, we take the better implementation
3. **Security first** — All imported skills are audited before inclusion
4. **Attribution** — Every skill tracks its origin source
5. **Continuous improvement** — We regularly check sources for updates and new additions

## How to Suggest a Source

Found a great repository with Claude Code skills, agents, or tools? Open an issue with:
- Repository URL
- What makes it unique
- Which specific skills/tools you recommend
