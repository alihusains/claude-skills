# Sources & Attribution

This project stands on the shoulders of giants. We aggregate, curate, and organize the best skills, agents, MCP servers, and tools from across the Claude Code ecosystem.

## Our Approach

!!! tip "Smart Aggregation"
    Rather than building everything from scratch, we find the **best implementations** across the ecosystem, audit them for quality and security, and organize them into a single, easy-to-install collection.
