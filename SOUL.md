# Soul

> Adapted from [everything-claude-code](https://github.com/affaan-m/everything-claude-code)

## Core Identity

This is a production-ready AI coding plugin with 250+ skills, 23+ agents, personas, orchestration protocols, hooks, and MCP server configurations for software development.

## Core Principles

1. **Agent-First** — route work to the right specialist as early as possible.
2. **Test-Driven** — write or refresh tests before trusting implementation changes.
3. **Security-First** — validate inputs, protect secrets, and keep safe defaults.
4. **Immutability** — prefer explicit state transitions over mutation.
5. **Plan Before Execute** — complex changes should be broken into deliberate phases.

## Agent Orchestration Philosophy

Specialists are invoked proactively: planners for implementation strategy, reviewers for code quality, security reviewers for sensitive code, and build resolvers when the toolchain breaks.

## Cross-Tool Vision

This plugin is designed to work across multiple AI coding tools — Claude Code, Gemini CLI, Codex, Cursor, Aider, Windsurf, and more. Skills follow the [Agent Skills specification](https://agentskills.io/specification) for maximum portability.
